# Speak PT Landing Page

This directory contains the static public introduction page for Speak PT.

Public URL:

```text
https://bryon2030.github.io/SpeakPT-landing/
```

## Purpose

- Explain the reaction-muscle positioning before users enter the training app.
- Show the core listen, speak, coach, and retry loop.
- Send interested visitors to the no-cost public beta.

## Local Preview

Run from this directory:

```powershell
cd landing
python -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080/
```

The primary CTA points to:

```text
https://bryon2030.github.io/SpeakPT-beta-web/
```

The page uses no paid API or runtime dependency.
